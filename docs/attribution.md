# Attribution

Spanish Law Research uses public Spanish legislation data from Legalize and official BOE sources.

## Sources

- Legalize: https://github.com/legalize-dev/legalize
- Spanish corpus: https://github.com/legalize-dev/legalize-es
- BOE: https://www.boe.es/

Legalize describes legislative content as public domain because it is sourced from official government publications. Legalize describes repository structure, metadata, and tooling as MIT-licensed.

## Required Attribution In This App

Public documentation and generated tool responses must preserve:

- Legalize project attribution.
- `legalize-dev/legalize-es` corpus attribution.
- BOE or official source URLs when present.
- Law identifier.
- Jurisdiction.
- Source file path.
- Source commit or revision.
- Last update date.

## Non-Endorsement

This app is not an official BOE service, public authority, court, law firm, or Legalize-maintained product. It provides source-grounded legal research support and does not provide legal advice.
